===========
FPDC Client
===========


This package provides an API to interact with `FPDC`_. It is based on
`coreapi`_ for the REST mechanisms.

.. _`FPDC`: https://github.com/fedora-infra/fpdc/
.. _`coreapi`: http://core-api.github.io/python-client/
